### New: ParseHTMLTables Pack v1.0.0 (Community Contributed)
#### Scripts
##### ParseHTMLTables
Find tables inside HTML and extract the contents into objects using the following logic:

- If the table has 2 columns and has no header row, treat the first column as key and second as value and create a table of key/value.
- If the table has a header row, create a table of objects where the attribute names are the headers.
- If the table does not have a header row, create a table of objects where the attribute names are cell1, cell2, cell3...

### New: PenfieldAI Pack v1.0.0 (Partner Supported)
#### Integrations
##### Penfield
The ***penfield-get-assignee*** command takes in necessary context data, and returns the analyst who Penfield believes the incident should be assigned to based on Penfield's models of skill and process. The ***test*** command verifies that the endpoint is reachable.

##### Penfield Assign

#### Scripts
##### PenfieldAssign
This script uses the Penfield integration's ***penfield-get-assignee*** command to determine who an incident should be assigned to. It then adds the selected analyst to the War Room and overwrites the owner property.
